# Content guidelines

- Labels should name the action: “Send inquiry,” not “Submit.”
- Buttons perform actions; links navigate.
- Error messages explain the problem and the next useful action.
- Avoid placeholder text as the only field instruction.
- Keep alt text contextual and concise. Use empty alt text for decorative images.
- Do not force short demo copy. Components must tolerate real headings, translations, and long labels.
- Do not place private website copy or client details in public examples.
