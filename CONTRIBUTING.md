# Contributing

1. Create a focused branch.
2. Keep APIs generic and typed strictly.
3. Add or update stories, interaction coverage, and accessibility guidance.
4. Add a Changeset for user-facing changes.
5. Run `pnpm check`.

All contributors and coding agents must follow [`AGENTS.md`](AGENTS.md). Public CSS uses stable `rvds` BEM classes, and every custom property begins with `--rvds-`. Accessibility requirements are release requirements, not optional cleanup.

Never add private website content, Squarespace exports, credentials, personal contact details, client materials, or unlicensed brand assets.
